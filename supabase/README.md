# Supabase Setup (Stage 2)

1. Create a free Supabase project.
2. Open SQL Editor in Supabase.
3. Run schema.sql from this folder.
4. In Authentication > Providers, keep Email enabled.
5. In Authentication > URL Configuration, add your local frontend URL:
   - http://localhost:5173

Copy project values into local env files:
- frontend/.env
- backend/.env
